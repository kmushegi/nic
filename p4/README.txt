The code in this repository contains an implemention of the Perceptron
artificial neuron model based neural network.

----------------------------------------

Repository structure

(D) = Directory

p4
	* data(D)
		- two data sets distributed by Professor Majercik
	* figures(D)
		- figures used in the final report
	* src(D)
		- auto_test.py
			- automated testing utility
			- relies on joblib package for multiprocessing
		- data_loader.py
			- reads, formats and loads data sets from paths defined in
			data_loader.py on lines 16-20
		- network.py
			- handle all of neural network logic
		- NNRunner.py
			- point of entry into the program
		- setup.py
			- a classic setup.py file with basic program information
	* stats(D)
		- files generated by auto_test.py
	* stats_nn.xlsx
		- files from /stats/ analyzed with MS Excel

----------------------------------------

The software allows and requires certain parameters for correct execution; the
format and ordering of parameters is specified below

**** Run ****
$ python NNRunner.py bit_map/downsampled (1/0) n_out_neurons learning_rate

If parameters are supplied incorrectly (length != 4) the program will run with
default parameters defined in NNRunner.py on lines 27-29.

-----------------------------------------

Automated testing was done using the auto_test.py Python script which runs
all possible (well, as defined by us, the almighty creators) combinations of 
parameters on the data provided by data_loader.py. auto_test.py runs several
instances of the NNRunner.py in parallel; current settings run #(cpu_cores) 
jobs.

The output is piped to .txt files in the 'stats' folder.

To run the script, type

$ python auto_test.py

-----------------------------------------

Data analysis was done using the MS Excel. More in stats_nn.xlsx.
